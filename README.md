🌐 Web de Programación

Plataforma educativa diseñada para enseñar conceptos fundamentales de programación, herramientas esenciales, buenas prácticas y rutas de aprendizaje para desarrolladores.

El sitio incluye contenido teórico, recursos educativos y una sección de tutoriales en video para facilitar el aprendizaje de forma visual.

📚 Contenido del sitio

La web está organizada en varias secciones educativas:

🏠 Inicio

Presentación de la plataforma y su objetivo: facilitar el aprendizaje de programación de forma clara y estructurada.

🎥 Tutoriales

Sección de videos educativos que explican conceptos de programación y desarrollo web.

📖 Conceptos

Explicación de fundamentos como:

Qué es HTML

Etiquetas HTML

Atributos

Estructura de un documento web
